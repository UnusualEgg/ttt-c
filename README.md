# Tic Tac Toe (in C)

my last one was in rust

## Usage
### Options
- Address: -a n.n.n.n
- Port: -p n
- Specify if server: -s

### Playing
First character can be any of the following but the second should be blank or 'l' or 'r'
- t=top
- l=left
- r=right
- b=bottom
- c=center
#### Example
tl=top left
c=center
br=bottom right
r=right
